# 🤖 Registro de Uso de IA

Para acelerar o desenvolvimento e garantir a aplicação de boas práticas, utilizei IA (Gemini) como um _pair programmer_. Abaixo, os marcos mais importantes:

- **Arquitetura e SOLID:** > "Refatorar o módulo de checkout para seguir os princípios SOLID. Aplicar Clean Architecture separando estritamente a camada de domínio (interfaces), aplicação (use-cases) e infraestrutura (Prisma/Fastify)."
- **Tratamento de Concorrência:** > "Como garantir a atomicidade no débito de estoque usando Prisma em uma transação? Preciso evitar condições de corrida em cenários de alta concorrência."

- **Resiliência (Rollback):** > "Desenvolver a lógica de ação compensatória para o `FakeERPService`. Se a integração com o ERP falhar, como reverter o débito de estoque no banco de forma atômica?"

- **Front-end e UX:** > "Criar um componente `ProductCard` responsivo com Tailwind v4 e Framer Motion. Adicionar estados de loading para evitar cliques duplicados e garantir a melhor experiência Mobile First."
